'@backstage/create-app': patch
---

Bumped the `typescript` version in the template to `~4.6.4`.

To apply this change to an existing app, make the following change to the root `package.json`:

```diff
   dependencies: {
     ...
-    "typescript": "~4.5.4"
+    "typescript": "~4.6.4"
   },
```
